# UEditorGUI
基于面向对象为Unity3D Editor封装了一套GUI框架，在简单的Editor扩展时不再过于关心控件的状态而集中于业务逻辑的开发。<br>
同时，提供了更丰富的基础控件：
* Label
* Button
* Field
* Toggle
* Popup
* Slider
* Helpbox
* Toolbar
* Drawer
* Listview
* Treeview

注：仅在 Unity 5 和 Unity 2017 部分版本上测试过
